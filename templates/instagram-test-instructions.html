Testing Instructions for Instagram Direct Integration

These steps explain how to test the instagram_manage_messages permission for this app.

Test accounts

Business Instagram account used for testing: @test_workflow

Please use the test user credentials provided in the Meta “Test Users” section to send messages during review.

How to test

  1. Open the Instagram app or web client and log in as the test user.
  2. Send a Direct Message to the business account @test_workflow with any short text (for example, “Hello”).
  3. Our n8n workflow receives the message via the Instagram Webhook, processes it and sends an automated reply back into the same conversation within a few seconds.
  4. You should see an automatic reply describing the parachute trip offer (trip details, price and basic safety information).
  5. (Optional) Send the word “pay” in the same conversation. The workflow will respond with a test Stripe checkout link. You do not need to complete a real payment for the review.
  6. All messages are also stored in our internal Notion database and backed up to Google Sheets for logging and support purposes only. These internal tools are not required for testing.

If you experience any issues during testing, please contact us using the email address provided in the app’s contact details.
